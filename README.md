Client Description:
The client is a small game studio that is aiming to hire 10 new employees this year across a range of positions ranging from entry level to experienced. They seek to welcome those interested in working in the games industry with an approachable looking website.


Changes from assignment 1:
I added a home page and a gallery to give the landing page more context. I generally kept the layout of my landing page the same, other than moving the apply button, my main call to action, to the top so that it is the first thing the user is presented with. This is more logical since the "why choose us?" section may not be necessary for those who have already decided to apply. I also changed the images to invoke a game company as opposed to generic stock images.


AI Documentation:
I did not use any external AI for this project as I wanted to use my own knowledge as well as the examples from our tutorials to deepen my understanding of HTML and CSS.
